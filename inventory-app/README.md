# inventory-app

Frontend Angular 18 (standalone + signals) de StockFlow. Ver el
[README raíz](../README.md) del monorepo para stack completo, requisitos,
comandos de arranque y tests.

Arranque rápido:

```bash
npm install
npm start
```

Levanta en `http://localhost:4200`. Requiere el backend corriendo en
`http://localhost:8080`.
