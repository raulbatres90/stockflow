# inventory-service

Backend Spring Boot 3.5 de StockFlow. Ver el [README raíz](../README.md) del
monorepo para stack completo, requisitos, comandos de arranque y tests.

Arranque rápido:

```bash
mvn spring-boot:run
```

Levanta en `http://localhost:8080`. Swagger en `/swagger-ui.html`, health en
`/actuator/health`, consola H2 en `/h2-console`.
